How It Works

A VR memory game that detects early Alzheimer's signs through immersive cognitive tasks

Comfortable Home Screening

Users complete familiar memory tasks in a 3D virtual environment using consumer VR headsets like Meta Quest.

  • Memorize objects, answer questions, find items again
  • Tracks accuracy, reaction time, and error rates
  • Multilingual voice guidance and simple controls
  • Cloud-based performance analysis
  • Designed for users 50+ with no VR experience needed
Healthy users: 1.5s avg response 1.5s
At-risk users: 2.5s+ avg response 2.5s+

Clinical Accuracy

Our game uses clinically-validated memory assessment principles transformed into an engaging experience.

The distraction phase between memory tasks mimics real-world cognitive challenges. Our proprietary scoring system tracks subtle performance changes.

Built with Unity and XR toolkits, featuring voice recognition, behavioral tracking, and longitudinal analysis.
